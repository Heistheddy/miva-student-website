# Student Portfolio & Academic Management Website

## Overview
A responsive, multi-page personal academic portfolio and student management platform developed for the **COS 106 - Introduction to Web Technologies** Term Project at MIVA Open University. This project showcases my academic journey, technical skills, and practical web development abilities.

## Features
- **Homepage:** Personal biography, welcome message, and profile photograph.
- **About Me:** Educational background, career aspirations, structured skills table, and certifications.
- **Projects Showcase:** A gallery of practical projects with descriptions and live links to web applications.
- **Academic Planner:** A fully interactive task management To-Do list built with Vanilla JavaScript (add, complete, delete functionality).
- **Contact Page:** A contact form with comprehensive JavaScript validation (empty field checks, email format validation, and numeric-only phone validation).

## Technologies Used
- **HTML5:** Semantic elements, forms, tables, and media integration.
- **CSS3:** Responsive design, Flexbox/Grid layouts, CSS variables, and interactive hover animations.
- **JavaScript (ES6):** DOM manipulation, event handling, state management (arrays), and dynamic validation.
